# Beverly Chemutai's Portfolio

![Website Preview](images/hero-bg.jpg)

A personal portfolio website showcasing Beverly Chemutai's web development skills and Kenyan culinary expertise.

## Features

- Responsive design for all devices
- Interactive profile section with clickable image
- Animated skill bars
- Two authentic Kenyan recipes
- Clean, modern design with Kenyan color scheme

## Pages

1. *Homepage* - Introduction and featured recipes
2. *About* - Interactive profile with skills
3. *Ugali Recipe* - Traditional Kenyan staple food
4. *Pilau Recipe* - Fragrant Kenyan spiced rice

## Technologies Used

- HTML5
- CSS3
- JavaScript
- Git

## How to Run

1. Clone this repository
2. Open index.html in your browser
3. Navigate through the different pages

## Image Requirements

Place these images in the images folder:
- hero-bg.jpg - Homepage background
- profile.jpg - Main profile picture
- profile-traditional.jpg - Traditional attire picture
- ugali.jpg - Ugali recipe image
- pilau.jpg - Pilau recipe image
